# lloyal-sdk

Backend-agnostic TypeScript SDK for lloyal inference primitives.

## Packages

| Package                      | Description                                                   |
| ---------------------------- | ------------------------------------------------------------- |
| `@lloyal-labs/sdk`           | Core primitives — Branch, BranchStore, Session, Rerank, types |
| `@lloyal-labs/lloyal-agents` | Structured concurrency agent runtime (Effection v4)           |

## Build

```bash
npm install
npm run build
```
